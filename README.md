# Pesquisa de Opinião da Tudo Web 

Este projeto em Python tem como objetivo desenvolver um Programa para Pesquisa de Opinião da empresa de marketing fictícia chamada Tudo Web. O propósito é: *Utilizar a estrutura de repetição FOR e estruturas de decisão para verificar a opinião do entrevistado*. O programa deve solicitar a digitação do nome, idade e opinião do entrevistado sobre o atendimento prestado pela empresa sendo: *1: EXCELENTE; 2: BOM e; 3: RUIM.*
O programa realiza a pesquisa com 50 entrevistados. A pesquisa de opinão foi obtida da seguinte forma: **for i in range (1, 50 + 1)**, onde a estrutura de repetição "for" coletar, no intervalo de 1 a 3 avaliações e exibindo e o retorno com a quantidade de respostas de atendimento excelente ao cliente e a quantidade de respostas ao atendimento ruim ao cliente.
Nas instruções de execução do programa, é soliciado ao usuário que insira o *nome* *idade* e *opinião*.
